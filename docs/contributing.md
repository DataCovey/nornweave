# Contributing

Thank you for your interest in contributing to NornWeave. Please read our [Contributing Guide](https://github.com/nornweave/nornweave/blob/main/CONTRIBUTING.md) and [Code of Conduct](https://github.com/nornweave/nornweave/blob/main/CODE_OF_CONDUCT.md) in the repository.

Summary:

- Use **uv** (or pip) and **make** for install, test, lint, format.
- Follow **Conventional Commits** and open a PR with a clear description.
- Add tests for new behavior and keep documentation up to date.
